# Hi, I'm José Riascos 👋

**PhD Economist · Data Analyst · Applied Researcher**

I work at the intersection of data engineering, causal inference, and public policy. Currently a Data Analyst in R&D at [Avidbots](https://www.avidbots.com), and a Research Affiliate at Université d'Orléans.

- **Causal inference** — DID, Staggered DID, IV, Event Study, PSM, TWFE, Synthetic DID
- **Data pipelines** — SQL, Python (Pandas, NumPy), R, Stata; end-to-end from raw API/satellite data to clean analytical datasets
- **Geospatial analysis** — QGIS, Google Earth Engine (GEE), spatial econometrics
- **Policy evaluation** — program impact assessment, indicator design, evidence-based recommendations
